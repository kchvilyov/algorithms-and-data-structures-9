Практическое задание 9
Существует метод test, который принимает ссылку типа List (с
любой допустимой параметризацией). Содержание данного метода для работы
несущественно, выберем логирующую заглушку.

Вызовите метод test, передав туда список с пятью числами.

После завершения метода test выведите в лог число —
количество, сколько раз (в сумме) вызывались любые перегрузки метода add.

Существенным условием является то, что test не может и не
должен знать ничего о сути задачи.


## Критерий приема
Для работы требуется Java 17.
Запуск тестов выполняется через `./gradlew test`. Задание считается выполненным, если все тесты успешно пройдены.
